CODING SOMETHING INTERACTIVE IN P5.JS

Use the number keys to build the perfect burger :)

[9] and [0] are the condiments, press [X] to clear the canvas and press [P] to save your creation!


For this assignment, we were given some code that functioned as a diy photoshop and the task was to customize the brushes. I chose to make a game where you are in the kitchen of a burger place and you're tasked with assembling a burger. The brushes were customized to be some of the classic burger ingredients. The cashier looking through the window is me. The self-portrait was actually coded in p5.js for another assignment that I pixelated and added to the background art. All of the burger ingredients (excluding the condiments) and background art were drawn in Procreate.

function draw() {
  if (keyIsPressed) {
    choice = key; // set choice to the key that was pressed
    clear_print(); // check to see if it is clear screen or save image
  }
  if (mouseIsPressed){
    newkeyChoice(choice);  // if the mouse is pressed call newkeyChoice
  }
}

function clear_print() {
// this will do one of two things, x clears the screen by resetting the background
// p calls the routine saveme, which saves a copy of the screen
  if (key == 'x' || key == 'X') {
    background(screenbg); // set the screen back to the background color
  } else if (key == 'p' || key == 'P') {
     saveme();  // call saveme which saves an image of the screen
  }
}

function saveme(){
    //this will save the name as the intials, date, time and a millis counting number.
    // it will always be larger in value then the last one.
  filename=initials+day() + hour() + minute() +second();
  if (second()!=lastscreenshot) { // don't take a screenshot if you just took one
    saveCanvas(filename, 'jpg');
    key="";
  }
  lastscreenshot=second(); // set this to the current second so no more than one per second
  
}
